||立体图形|长方体、正方体、圆柱的表面积公式；长方体、正方体、圆柱、圆锥的体积公式；体积 / 容积单位换算|圆柱表面积分清是否需要加底面积；圆锥体积计算勿忘 ×1/3；相邻体积单位进率为 1000|
||图形变换与位置|轴对称、平移、旋转三要素；图形放大与缩小；数对（先列后行）；方向 + 距离确定位置|旋转三要素：旋转中心、方向、角度；数对（列，行）切勿写反；方位判断必须找准观测点|
|**统计与概率（占比 10%）**|统计核心|三类统计图特点（条形看数量、折线看变化、扇形看占比）；平均数、中位数、众数；用样本估计总体|平均数易受极端值影响；分析统计图先看清坐标轴刻度与单位|
||概率核心|事件分类（必然 / 不可能 / 随机事件）；用分数表示可能性大小；游戏公平性判断；列表 / 树状图求概率|用频率估计概率的前提是大量重复试验|
|**核心公式速记**|通用必背|圆：C=2πr=πd，S=πr² 圆柱：S 表 = 2πr²+2πrh，V=Sh 圆锥：V=1/3Sh 行程：路程 = 速度 × 时间 浓度：浓度 = 溶质 ÷ 溶液 ×100%|所有公式应用前必须统一单位|

# 二、北师大版小学数学 1-6 年级每学期单元知识点预习清单

（严格贴合北师大版教材单元编排，预习聚焦核心知识点，适配课前预习、假期提前学）

## 一年级（启蒙奠基学段）

### 一年级上册

1. 生活中的数：10 以内数的认识、读写、大小比较，0 的意义与读写
    
2. 比较：比多少、长短、高矮、轻重，掌握基础比较方法
    
3. 加与减（一）：10 以内加减法的意义，口算方法，解决简单实际问题
    
4. 分类：按单一 / 自定标准分类，培养有序思维
    
5. 位置与顺序：认识前后、上下、左右，建立基础空间方位感
    
6. 认识图形：直观认识长方体、正方体、圆柱、球 4 种立体图形
    
7. 加与减（二）：20 以内数的认识（个位、十位），20 以内进位加法
    
8. 认识钟表：认读整时、半时，了解时间基础概念
    

### 一年级下册

1. 加与减（一）：20 以内退位减法，解决比多比少实际问题
    
2. 观察物体：从正面、侧面、上面观察简单物体，辨认观察到的形状
    
3. 生活中的数：100 以内数的认识、读写、大小比较，认识百位
    
4. 有趣的图形：认识长方形、正方形、三角形、圆，了解七巧板
    
5. 加与减（二）：100 以内两位数加减一位数、整十数口算
    
6. 加与减（三）：100 以内两位数加减两位数竖式计算，连加连减混合运算
    
7. 数学好玩：分扣子、填数游戏，培养有序思考与逻辑推理能力
    

## 二年级（乘除入门学段）

### 二年级上册

1. 加与减：100 以内连加、连减、加减混合竖式计算
    
2. 购物：认识人民币元、角、分，简单换算与购物计算
    
3. 数一数与乘法：乘法的意义，乘法各部分名称，同数连加与乘法的关系
    
4. 图形的变化：感知轴对称、平移、旋转现象，基础图形操作
    
5. 2\5 的乘法口诀：2\5 乘法口诀，用口诀口算表内乘法
    
6. 测量：认识长度单位厘米、米，单位换算，物体长度测量
    
7. 分一分与除法：平均分的意义，除法的意义，用乘法口诀求商
    
8. 6\9 的乘法口诀：6\9 乘法口诀，表内乘法全覆盖
    
9. 除法：除法的两种意义，倍的初步认识，乘除法互逆关系
    

### 二年级下册

1. 除法：有余数的除法，除法竖式，掌握「余数必须小于除数」核心规则
    
2. 方向与位置：认识 8 个基本方位，能描述简单路线图
    
3. 生活中的大数：万以内数的认识、读写、大小比较，认识千、万计数单位，近似数
    
4. 测量：认识分米、毫米、千米，全长度单位换算
    
5. 加与减：三位数加减法口算与笔算，连加连减混合运算
    
6. 认识图形：认识角、直角、锐角、钝角，掌握长方形、正方形特征，平行四边形初步
    
7. 时、分、秒：时、分、秒的认识与换算，认读钟面时刻，简单时间计算
    
8. 调查与记录：数据收集与整理方法，画「正」字记录数据，认识简单统计表
    

## 三年级（运算进阶学段）

### 三年级上册

1. 混合运算：整数四则混合运算顺序，先乘除后加减，有括号先算括号内
    
2. 观察物体：从三个方向观察立体图形，辨认观察到的平面形状
    
3. 加与减：三位数加减混合运算，里程表类实际问题解决
    
4. 乘与除：两、三位数乘 / 除以一位数口算，乘除法估算
    
5. 周长：周长的定义，长方形、正方形周长计算公式与实际应用
    
6. 乘法：两、三位数乘一位数笔算，多位数乘一位数进阶
    
7. 年、月、日：年、月、日的认识，平年闰年判断，24 时计时法，简单时间计算
    
8. 认识小数：小数的初步认识、读写、大小比较，一位小数加减法
    

### 三年级下册

1. 除法：两、三位数除以一位数笔算，商中间 / 末尾有 0 的除法，除法估算
    
2. 图形的运动：轴对称、平移、旋转进阶，方格纸上规范作图
    
3. 乘法：两位数乘两位数口算与笔算，乘法估算
    
4. 千克、克、吨：认识质量单位克、千克、吨，掌握单位换算
    
5. 面积：面积的定义，面积单位认识与换算，长方形、正方形面积计算
    
6. 认识分数：分数的初步认识，分数的意义，同分母分数大小比较与加减法
    
7. 数据的整理和表示：分类整理数据，简单条形图绘制与数据分析
    

## 四年级（逻辑构建学段）

### 四年级上册

1. 认识更大的数：亿级、万级大数的读写、改写，十进制计数法，四舍五入求近似数
    
2. 线与角：线段、射线、直线，平行与垂直，角的度量，平角、周角，量角与画角
    
3. 乘法：三位数乘两位数笔算，大数估算，计算器的使用
    
4. 运算律：加法 / 乘法五大运算律，简便运算方法全掌握
    
5. 方向与位置：用数对确定位置，用方向 + 距离确定物体位置
    
6. 除法：三位数除以两位数笔算，商不变的规律，行程问题核心公式
    
7. 生活中的负数：认识正负数，0 的中性意义，正负数的生活应用
    
8. 可能性：事件发生的确定性与不确定性，可能性大小判断
    

### 四年级下册

1. 小数的意义和加减法：小数的意义、性质，小数大小比较，小数加减法与混合运算
    
2. 认识三角形和四边形：三角形分类，三角形内角和、三边关系，平行四边形、梯形的特征
    
3. 小数乘法：小数乘法的意义，小数乘整数、小数乘小数笔算，小数混合运算与简便运算
    
4. 观察物体：从不同方向观察立体图形，根据平面图形还原立体图形
    
5. 认识方程：用字母表示数，等量关系，方程的意义，等式的性质，解简易方程
    
6. 数据的表示和分析：条形统计图、折线统计图，平均数的意义与计算
    
7. 数学好玩：优化思想（沏茶、烙饼问题）、密铺
    

## 五年级（体系完善学段）

### 五年级上册

1. 小数除法：小数除法笔算，商的近似数，循环小数，小数四则混合运算与简便运算
    
2. 轴对称和平移：轴对称图形的性质，平移的应用，方格纸上图案设计
    
3. 倍数与因数：因数与倍数，2、3、5 的倍数特征，奇数与偶数、质数与合数，公因数与公倍数
    
4. 多边形的面积：平行四边形、三角形、梯形的面积计算公式，组合图形面积计算
    
5. 分数的意义：分数的意义，分数与除法的关系，真分数 / 假分数，分数基本性质，约分与通分
    
6. 组合图形的面积：组合图形面积计算，不规则图形面积估算
    
7. 可能性：用分数表示可能性大小，判断游戏公平性，设计公平游戏方案
    

### 五年级下册

1. 分数加减法：异分母分数加减法，分数加减混合运算，分数与小数的互化
    
2. 长方体（一）：长方体、正方体的特征，展开与折叠，表面积计算公式与应用
    
3. 分数乘法：分数乘法的意义与计算方法，倒数的认识，分数乘法实际应用
    
4. 长方体（二）：体积与容积的定义，体积单位换算，长方体、正方体体积计算，不规则物体体积测量
    
5. 分数除法：分数除法的意义与计算方法，分数除法实际应用，用方程解决分数问题
    
6. 确定位置：用方向 + 距离精准确定位置，描述复杂路线图
    
7. 用方程解决问题：解 ax±x=b 类型方程，用方程解决相遇问题
    
8. 数据的表示和分析：复式条形统计图、复式折线统计图，平均数进阶应用
    

## 六年级（综合拔高・小升初衔接学段）

### 六年级上册

1. 圆：圆的特征，圆心、半径、直径，圆的周长、面积计算公式，扇形初步认识
    
2. 分数混合运算：分数四则混合运算，简便运算，复杂分数应用题
    
3. 观察物体：从三个方向观察立体图形，判断观察范围
    
4. 百分数：百分数的意义、读写，百分数与小数、分数的互化，基础百分数应用题
    
5. 数据处理：扇形统计图，三类统计图的特点与选择，综合数据分析
    
6. 比的认识：比的意义、基本性质，比的化简，按比分配实际应用
    
7. 百分数的应用：百分数进阶应用题（增长率、浓度、利率、折扣）
    

### 六年级下册

1. 圆柱与圆锥：圆柱、圆锥的特征，圆柱表面积计算，圆柱、圆锥体积计算公式与应用
    
2. 比例：比例的意义、基本性质，解比例，比例尺的认识与应用
    
3. 图形的运动：平移、旋转、轴对称综合应用，图形的放大与缩小
    
4. 正比例与反比例：正比例、反比例的意义，正反比例判断与实际应用
    
5. 总复习：小学阶段数与代数、图形与几何、统计与概率全体系知识整合，小升初考点复盘
